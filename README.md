<h2>Getting Started with files</h2>

To get started you can simply clone the repo and install the dependencies in the root `folder`

| Steps   | with [NPM](https://www.npmjs.com/) |
| ------- | ---------------------------------- |
| Install | `npm install`                      |
| Enterfolder | `cd backend`                   |
| Run     | `npm run dev`                      |
